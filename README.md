# everyMail Api Server

명지대학교 자연캠퍼스 인터넷 프로그래밍 팀 프로젝트 server

---

### Requirements

```
- Node.js
- yarn
```

---

### How to start development server

```
yarn install
yarn dev
```

---

### Commit conventions

```
- Feat: 새로운 기능을 추가할 경우
- Fix: 버그를 고친경우
- !HOTFIX: 급하게 치명적인 버그를 고쳐야하는 경우
- Style: 코드 포맷 변경, 코드 수정을 하는경우
- Refactor: 리팩토링하는 경우
- Chore: 그 외
```
